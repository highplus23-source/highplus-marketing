# 닥터포스트(Doctor Post) 랜딩페이지 분석 보고서

## 1. 핵심 타겟 및 메인 카피
- **메인 카피**: "원장님은 진료에만 집중하세요."
- **타겟 페인포인트(Pain Point)**: 대행사는 비싸고, 직접 하자니 시간이 없고, 직원에게 맡기자니 퀄리티가 걱정되는 원장님들의 마케팅 고충.

## 2. 3대 차별화 포인트 (The 3 NOs)
1. **NO Study (학습 불필요)**: 프롬프트 공부나 마케팅 지식 불필요. 전문 매니저가 세팅.
2. **NO Install (설치 없음)**: PC에 잡다한 프로그램 설치 없이, 자체 인프라를 원격으로 구축.
3. **NO Risk (보안/해킹 위험 제로)**: 자체 하드웨어(닥터박스)를 통해 병원 내부 IP를 사용하여 네이버 계정 해킹이나 저품질 블로그가 될 위험 차단. (비밀번호 제공 불필요)

## 3. 주요 기술 및 서비스 (닥터 4종 세트)
- **Dr. Memory (지능형 학습)**: 원장님의 말투와 진료 철학을 AI가 학습해 6개월 뒤 94%의 문체 일치율 달성.
- **Dr. Shield (의료법 자동 방패)**: AI가 금칙어를 실시간 필터링하여 보건소 신고 등 의료법 위반 사전 차단 (2026년 AI 표기 의무화 자동 대응).
- **Dr. Canvas (AI 의료 비주얼)**: 전문 디자이너 없이도 본문에 맞는 의료 일러스트/배너 자동 생성.
- **Dr. Graph (자산화)**: 대행사 계약 종료 시 글이 날아가는 방식이 아닌, 병원만의 영구적인 지식 자산(데이터베이스)으로 축적.

## 4. 핵심 하드웨어 전략: '닥터박스'
- **방식**: LAN 선과 전원만 꽂으면 60초 내 연결되는 스탠드얼론 무인 기기.
- **효과**: 원장님 PC가 꺼져있거나 주말에도 병원 내부 청정 IP를 통해 24시간 자동 발행. 비밀번호 공유라는 심리적 허들을 하드웨어로 해결.

## 5. 소셜 프루프 (Social Proof)
- 전국 100명 이상의 원장님이 사용 중이라는 데이터 제시.
- 대행사와의 분명한 선 긋기("매달 수백만 원을 태워도 남는 게 없는 임대형 마케팅은 이제 그만하세요").

# 💡 하이플러스 마케팅 핵심 철학 (vs 닥터포스트)

닥터포스트가 '100% 완전 자동화(Zero-Effort)'를 강조한다면, **하이플러스는 '원장님의 진정성'을 지키면서 '노력'만 최소화하는 하이브리드 전략**을 취합니다. 대표님이 직접 9년간 병원을 운영하며 현장에서 깨달은 철학이 가장 강력한 무기입니다.

### 1. "완전 자동화는 마약이자 독이다" (Anti-Full Automation)
- **완전 자동화의 맹점**: 원장님이 블로그 내용을 전혀 모른 채 환자를 맞이하면, 신뢰도(Rapport)가 급격히 하락합니다. 대형 병원이 아닌 이상, 환자는 블로그의 '원장님 글'을 보고 내원하기 때문입니다.
- **하이플러스의 철학**: 내 이름표를 달고 나가는 글은 원장님이 반드시 숙지하고 있어야 합니다. 원장님을 마케팅에서 '완전 배제'시키는 것이 아니라, 참여하기 쉽게 '허들'을 대폭 낮춰줍니다.

### 2. "원장님들은 사실 본인 이야기를 쓰고 싶어 한다"
- **현장의 목소리**: 바쁜 진료 때문에 못 쓸 뿐이지, 원장님들은 본인의 진료 철학과 생생한 임상 케이스를 직접 담고 싶어 합니다.
- **하이플러스의 해결책**: 초안 작성, 법적 위험(의료법) 검토, 이미지 디자인 등 시간과 에너지가 많이 드는 작업을 시스템이 처리하여, 원장님은 오직 '원장님만이 할 수 있는 핵심 메시지 전달'에 집중하도록 돕습니다.

### 3. 실무 중심 협업 구조 (원장님 + 마케팅 실장)
- 병원 내 마케팅 직원이 전체 글을 전담하면 의료 전문성이 크게 떨어집니다.
- **하이플러스 핏(Fit)**: 
  - **원장님**: 시스템의 지원(전문지식 초안)을 받아 글의 방향성을 잡고 최종 퇴고만 진행.
  - **마케팅 직원**: 발행 일정(스케줄) 관리 및 단순 업로드 등 오퍼레이팅 수행.

**결론 및 향후 홈페이지 기획 방향**: 
웹마케팅 사이트 메인 카피에 **"원장님의 이야기가 빠진 자동화는 독입니다. 진정성은 살리고, 노력은 1/10로 줄여드립니다"**와 같은 메시지를 전면에 내세우겠습니다. 완전 자동화의 함정을 정확히 찌르면서원장의 진심을 가장 편하게 꺼내어주는 완벽한 도구로 포지셔닝하여 기획을 진행하겠습니다.
