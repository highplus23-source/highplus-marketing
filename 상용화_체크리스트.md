# HighPlus 홈페이지 상용화 체크리스트

## 현재 상태 요약

| 항목 | 상태 | 비고 |
|------|------|------|
| Netlify 배포 | ✅ 완료 | git push → 자동 배포 |
| 보안 헤더 (`_headers`) | ✅ 완료 | HSTS, X-Frame-Options 등 |
| robots.txt | ✅ 완료 | 전체 허용 + sitemap 연결 |
| sitemap.xml | ✅ 완료 | 홈페이지 1개 등록 |
| 네이버 사이트 인증 | ✅ 완료 | 메타태그 삽입됨 |
| 폼 제출 (Web3Forms) | ✅ 완료 | 문의 접수 작동 중 |
| favicon | ✅ 완료 | favicon.svg |
| JSON-LD 구조화 데이터 | ✅ 완료 | WebSite, Organization, FAQPage 등 |
| OG 이미지 | ❌ **누락** | og-image.jpg 파일이 없음 |
| Google Analytics | ❌ 미설치 | 방문자 추적 불가 |
| og:image 메타태그 | ❌ **누락** | SNS 공유 시 이미지 안 뜸 |

---

## Phase 1. 출시 전 필수 작업

### 1-1. OG 이미지 생성 및 등록
> SNS(카카오톡, 네이버 등)에서 링크 공유 시 미리보기 이미지

- [ ] `img/og-image.jpg` 제작 (1200×630px 권장)
- [ ] `index.html`에 `<meta property="og:image">` 태그 추가
- [ ] 카카오톡 공유 테스트 → [카카오 디버거](https://developers.kakao.com/tool/debugger/sharing)
- [ ] 네이버 공유 테스트 → [네이버 웹마스터 도구](https://searchadvisor.naver.com/)

### 1-2. Google Analytics 설치
> 방문자 수, 유입 경로, 전환율 등 핵심 데이터 추적

- [ ] [Google Analytics 4](https://analytics.google.com/) 계정 생성
- [ ] 측정 ID (G-XXXXXXXXXX) 발급
- [ ] `index.html` `<head>`에 GA4 스크립트 삽입
- [ ] 실시간 보고서에서 트래킹 확인

### 1-3. Google Search Console 등록
> 구글 검색에 사이트가 제대로 노출되는지 관리

- [ ] [Google Search Console](https://search.google.com/search-console) 접속
- [ ] 도메인 소유권 인증 (DNS TXT 레코드 또는 HTML 메타태그)
- [ ] sitemap.xml 제출 (`https://highplus.work/sitemap.xml`)
- [ ] 색인 생성 요청

### 1-4. 네이버 서치어드바이저 최적화
> 네이버 검색 노출 관리 (이미 인증 완료)

- [ ] [네이버 서치어드바이저](https://searchadvisor.naver.com/) 로그인
- [ ] 사이트맵 제출 확인
- [ ] 웹 페이지 수집 요청
- [ ] 검색 노출 현황 확인

### 1-5. 폼 수신 테스트
> 실제 문의가 정상적으로 들어오는지 확인

- [ ] 테스트 문의 제출 → 이메일 수신 확인
- [ ] 필수 필드 미입력 시 에러 처리 확인
- [ ] 모바일에서 폼 제출 테스트
